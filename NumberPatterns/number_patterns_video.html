Number patterns
The hidden order
in numbers
Six patterns that mathematicians have marveled at for centuries.
Pattern 1 — Fibonacci sequence
Each term is the sum of the two before it. Found in sunflowers, shells, and galaxy spirals.
F(n) = F(n-1) + F(n-2)
Pattern 2 — Triangular numbers
Count the dots: 1, 3, 6, 10 — each adds the next row.
T(n) = n(n+1) / 2
Pattern 3 — Powers of 2
Double each time. The foundation of binary, computing, and exponential growth.
a(n) = 2ⁿ
Pattern 4 — Perfect squares
The difference between consecutive squares is always an odd number: 3, 5, 7, 9…
a(n) = n² → differences: 2n − 1
Pattern 5 — Prime numbers
No formula predicts the next prime — yet they never truly run out. Their density thins, but primes go on forever.
There are ∞ primes (Euclid, ~300 BC)
Pattern 6 — Collatz conjecture
Start at 27: if even ÷2, if odd ×3+1. Every number ever tested reaches 1. Nobody has proven this is always true.
if n even: n/2 · if n odd: 3n+1
