01 / 06 Commutative Property
Order doesn't matter — you can swap the numbers and still get the same result. Works for both addition and multiplication.
a + b = b + a   |   a · b = b · a
3 + 5 = 5 + 3 = 8
4 × 7 = 7 × 4 = 28
02 / 06 Associative Property
Grouping doesn't matter — you can regroup numbers with parentheses without changing the result.
(a + b) + c = a + (b + c)
(2 + 3) + 4 = 2 + (3 + 4) = 9
(2 · 3) · 4 = 2 · (3 · 4) = 24
03 / 06 Distributive Property
Multiplication distributes over addition. You can multiply each term inside the parentheses separately.
a(b + c) = ab + ac
3(4 + 5) = 3·4 + 3·5 = 27
2(x + 6) = 2x + 12
04 / 06 Identity Property
Every number has an identity element that leaves it unchanged. 0 is the additive identity; 1 is the multiplicative identity.
a + 0 = a   |   a · 1 = a
7 + 0 = 7
99 × 1 = 99
05 / 06 Inverse Property
Every number has an opposite (additive inverse) and a reciprocal (multiplicative inverse) that bring it back to the identity element.
a + (−a) = 0   |   a · (1/a) = 1
5 + (-5) = 0
3 × (1/3) = 1
06 / 06 Zero Product Property
If the product of two numbers is zero, then at least one of the factors must be zero. This is powerful in equation solving.
If a · b = 0, then a = 0 or b = 0
x(x − 3) = 0 → x = 0 or x = 3

Quick Reference
